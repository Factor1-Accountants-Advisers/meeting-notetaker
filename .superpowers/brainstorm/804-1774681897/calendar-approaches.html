C — Smart Strip (Only Days With Meetings)

Like Option A, but the strip only shows days that have meetings — no empty days cluttering the view. Dot count indicates number of meetings. "Today" is always shown. Closest to Vinyl's pattern with a cleaner feel for sparse calendars.
